# 🎬 NDUASHEY MAINA - CV Portfolio

A **Matrix-themed** interactive CV website hosted on GitHub Pages with live animations, techy fonts, and cyberpunk aesthetic.

## 🌐 Live Demo

Your CV will be available at: **`https://YOUR-USERNAME.github.io/cv`**

## ✨ Features

- 🟢 **Matrix Theme** - Dark background with neon green terminal aesthetic
- 📡 **Live Animations** - Falling matrix characters, glitch effects, pulsing boxes
- ⌨️ **Monospace Typography** - Techy Courier New font throughout
- 📦 **Boxy Design** - Grid-based layout with animated borders
- 💻 **Responsive** - Works on desktop, tablet, and mobile devices
- ⚡ **Zero Dependencies** - Pure HTML/CSS/JavaScript, no frameworks needed

## 🚀 Quick Start

### Step 1: Create a GitHub Repository

1. Go to [github.com/new](https://github.com/new)
2. Create a new repository named: **`cv`** (or any name you prefer)
3. Add description: "Matrix-themed interactive CV"
4. Make it **Public** (required for GitHub Pages)
5. Click **Create Repository**

### Step 2: Clone & Setup Locally

```bash
git clone https://github.com/YOUR-USERNAME/cv.git
cd cv
```

### Step 3: Add the Files

Copy `index.html` and this `README.md` to your repository:

```bash
# Your index.html is already in the project
# Just make sure it's in the root directory
```

### Step 4: Push to GitHub

```bash
git add .
git commit -m "Initial commit: Matrix-themed CV website"
git push origin main
```

### Step 5: Enable GitHub Pages

1. Go to your repository on GitHub
2. Navigate to **Settings** → **Pages**
3. Under "Source", select **Deploy from a branch**
4. Select branch: **main** and folder: **/ (root)**
5. Click **Save**

Your CV will be live in ~2 minutes at: **`https://YOUR-USERNAME.github.io/cv`**

## 📝 Customization

### Update Your Information

Edit `index.html` and update:
- Name and title in the header
- Contact information
- Experience entries
- Skills and certifications
- Links

### Change Colors

Search for these color codes in `index.html` and replace:
- `#00ff00` - Primary neon green
- `#00cccc` - Cyan accents
- `#ffff00` - Yellow highlights
- `#ff00ff` - Magenta accents

### Modify Animations

- **Matrix Rain Speed**: Adjust `duration` in the JavaScript section
- **Glitch Effect**: Modify `@keyframes glitch` in the CSS
- **Scanlines Speed**: Change `animation: scanlines 0.15s` value

## 🔗 Custom Domain (Optional)

To use a custom domain (e.g., `mycv.com`):

1. Update your domain's DNS settings to point to GitHub Pages
2. Go to repo Settings → Pages → Custom Domain
3. Enter your domain and save

[GitHub Pages Custom Domain Guide](https://docs.github.com/en/pages/configuring-a-custom-domain-for-your-github-pages-site)

## 🛠️ Troubleshooting

**Site not showing up?**
- Wait 5-10 minutes for GitHub Pages to deploy
- Check Settings → Pages to confirm it's enabled
- Ensure repository is **Public**

**Content not updating?**
- Clear your browser cache (Ctrl+F5 or Cmd+Shift+R)
- Give GitHub Pages 1-2 minutes to rebuild after push
